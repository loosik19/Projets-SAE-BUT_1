# Impl-mentation-d-un-besoin-client
Ce projet consiste à développer un programme permettant de répartir automatiquement les élèves de Poudlard dans les différentes maisons en fonction de leurs réponses à un questionnaire.  L’objectif est de remplacer le célèbre Choixpeau magique par un système informatique capable d’analyser les réponses des élèves et de déterminer leur maison.
