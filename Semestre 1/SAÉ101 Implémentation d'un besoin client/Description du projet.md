# Le nouveau choixpeau pas magique de Poudlard

Projet réalisé dans le cadre de la SAE 1.01 à l’Université Sorbonne Paris Nord.

## Description

Ce projet consiste à développer un programme permettant de répartir automatiquement les élèves de Poudlard dans les différentes maisons en fonction de leurs réponses à un questionnaire.

L’objectif est de remplacer le célèbre Choixpeau magique par un système informatique capable d’analyser les réponses des élèves et de déterminer leur maison :

- Gryffondor
- Serdaigle
- Poufsouffle
- Serpentard

Le programme utilise des scores attribués à plusieurs questions afin d’effectuer une répartition cohérente des élèves selon leurs qualités principales. :contentReference[oaicite:1]{index=1}

## Fonctionnalités

Le programme permet :

- de lire les réponses des élèves depuis un fichier ;
- d’extraire les informations des élèves ;
- de déterminer automatiquement la maison correspondant aux réponses ;
- de générer une répartition complète des élèves ;
- de comparer les résultats avec les affectations officielles du Choixpeau magique ;
- de calculer le pourcentage d’erreurs de la méthode utilisée ;
- de tester une répartition aléatoire afin de comparer les performances.

## Fonctionnalités principales

- Lecture et traitement de fichiers texte
- Utilisation de dictionnaires et de tableaux
- Répartition automatique des élèves
- Comparaison de résultats
- Gestion des données JSON
- Mise en place de tests unitaires
- Organisation du code en plusieurs fonctions

## Technologies utilisées

- Python
- Fichiers texte (`.txt`)
- Fichiers JSON (`.json`)
- Exécution dans le terminal
