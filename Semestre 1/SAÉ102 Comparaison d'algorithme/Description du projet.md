# Le nouveau choixpeau pas magique de Poudlard – SAE S1.02

Projet réalisé dans le cadre de la SAE S1.02 à l’Université Sorbonne Paris Nord.

## Description

Ce projet est la suite de la SAE S1.01 et consiste à améliorer un système de répartition automatique des élèves de Poudlard à l’aide de méthodes inspirées du machine learning.

Le programme analyse les réponses des élèves à un questionnaire de 10 questions afin de déterminer la maison qui leur correspond le mieux :

- Gryffondor
- Serdaigle
- Poufsouffle
- Serpentard

Pour effectuer cette répartition, plusieurs méthodes de prédiction ont été implémentées :

- la distance euclidienne ;
- la méthode du plus proche voisin ;
- la méthode des k plus proches voisins.

Le projet compare également la précision des différentes méthodes afin de déterminer laquelle reproduit le mieux les choix du véritable Choixpeau magique. 

## Fonctionnalités

Le programme permet :

- de lire des réponses depuis des fichiers texte ;
- de stocker les données dans des dictionnaires Python ;
- de calculer des distances euclidiennes entre élèves ;
- de déterminer automatiquement la maison d’un élève ;
- d’utiliser la méthode des k plus proches voisins ;
- de comparer les résultats avec les affectations officielles ;
- de calculer le pourcentage d’erreurs des différentes méthodes ;
- de réaliser des tests unitaires sur les fonctions principales.

## Fonctionnalités principales

- Lecture de fichiers `.txt` et `.json`
- Manipulation de listes et dictionnaires
- Algorithmes de comparaison de données
- Calculs mathématiques de distances
